Moment 1 - Publicering

Github Pages

För uppgiften har jag valt att ladda upp med Github pages, detta är ett kostnadsfri publiceringsplattform. För att kunna publicera på github pages måste man ha ett repo i github, där man sedan i inställningarna ställer in att sidan ska laddas upp i github pages.

Fördelar med github pages är att det är gratis att publicera hemsidan, hemsidan har god säkerhet och har en snabb CDN

Nackdelar med github pages är att den är statiskt vilket är den största nackdelen, det du ser är det du får vilket kan ställa till det om du till exempel vill ha en hemsida som ska interagera med en användare. En annan nackdel är att github pages stödjer HTML, CSS och JS. Det betyder att flera webbutveckling språk inte går att använda på github pages
