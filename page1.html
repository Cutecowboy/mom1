Moment 1 - Bootstrap ramverk

Om Bootstrap

Bootstrap är ett av de mest populära CSS ramverk. Bootstrap används för att designa en responsiv webbsida lättare än vanlig CSS. Bootstrap är designat utifrån "mobile first" där antagandet är att designen ska vara för mindre skärmar. Bootstrap fungerar som ett grid system med 12 kolumner där kolumnerna kan användas för att placera föremål i önskvärd plats eller bredd. Bootstrap kan användas genom att använda fördefinerade klasser i elementen för att komma åt den önskvärda designen.

Fördelar:

  • Man har kontroll över vilka element som ska designas
  • Det krävs mindre CSS kod för att få en responsiv hemsida
  • Man kan få flera funktionaliteter som dropdown, navbars, formulär etc. som är fördesignade
  • JavaScript och CSS kod är förprogrammerad vilket minskar utvecklingstiden
  • Gridbaserat som gör det lättare att placera föremål

Nackdelar:

  • Det kräver inte mycket kodkompetens för att designa sidan vilket kan bidra till att back-end utvecklare aldrig lär sig designa
  • Hemsidor kan få liknande utseende
  • Bootstrap CDN laddar hem alla funktionaliteter, även de som inte används
