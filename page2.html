Om Tailwind

Tailwind är ett CSS framework som är till för att underlätta utvecklarens designprocess genom att vara väldigt anpassningsbart, där alla byggstenar ges till utvecklaren. Så länge utvecklaren kommer över tröskeln att lära sig Tailwind blir det snabbare och lättare då man kan använda många kommandon/style för olika element.

Fördelar:

  • Du behöver inte koda i CSS, skapa egna klasser eller id
  • Minimerar CSS kod / Minimerar DRY
  • Du kan anpassa designen snabbt med alla verktyg som ges
  • Får en responsiv hemsida

Nackdelar:

  • All design görs i HTML, det kan ibland bli svårt och identifiera pga designkod.
  • Inlärningskurva, det tar tid och lära sig, även med cheatsheet behövdes en del googling
  • Ingen "färdig" mall, med detta menar jag t.ex. navbaren som behövde designas från grunden, bootstrap räckte med en class
